JWT kya hota hai?

JSON Web Token ek secure way hai jisme hum user ki information ko token ki form mein store aur transfer karte hain.

Simple:

  • JWT = login ke baad milne wala digital ID card

JWT kyu use hota hai?

  • User ko verify (authenticate) karne ke liye
  • Bar bar password dalne ki zaroorat nahi hoti
  • Secure communication ke liye

Example:

Jab tum kisi website par login karti ho → server tumhein JWT deta hai → phir tum har request ke sath wo token bhejti ho

JWT ke 3 parts hote hain

JWT 3 cheezon ka bana hota hai:

  • Header
    Batata hai token ka type aur algorithm
  • Payload
    Isme user ki info hoti hai (jaise user id, email)
  • Signature
    Security ke liye hota hai (token ko verify karta hai)

JWT ke faide (Advantages)

  • Fast & lightweight
  • Secure (tamper-proof)
  • Server ko session store nahi karna parta
  • APIs mein bohat use hota hai
