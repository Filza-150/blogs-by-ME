Bcrypt kya hota hai?

Bcrypt ek library hai jo passwords ko secure banane ke liye use hoti hai. Yeh passwords ko hash karta hai taake unhein direct read na kiya ja sake.

Simple:

  • Bcrypt = password ko secure code mein convert karne ka tool

Bcrypt kyu use hota hai?

  • Password ko safe rakhne ke liye
  • Hackers se protection ke liye
  • Plain text password store karne se bachne ke liye

Example:

Jab user signup karta hai → password ko direct save karne ke bajaye bcrypt se hash kiya jata hai → database mein hashed password save hota hai

Bcrypt ke main concepts

  • Hashing
    Password ko unreadable format mein convert karta hai
  • Salt
    Extra security layer jo password ko aur strong banati hai
  • Compare
    Login ke time entered password ko hashed password se match karta hai

Bcrypt ke faide (Advantages)

  • Strong security provide karta hai
  • Passwords ko safe rakhta hai
  • Brute force attacks se protection deta hai
  • Authentication systems mein bohat use hota hai
