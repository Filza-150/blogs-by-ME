Server Side Rendering (SSR) kya hota hai?

Server Side Rendering ek technique hai jisme web page server par generate hota hai aur phir browser ko ready-made HTML ke form mein bheja jata hai.

Simple:

  • SSR = page pehle server banata hai, phir user ko dikhata hai

SSR kyu use hota hai?

  • Fast initial page load ke liye
  • SEO (search engine optimization) improve karne ke liye
  • Better performance dene ke liye

Example:

Jab tum Google par koi website open karti ho → server pehle hi pura page bana kar bhej deta hai → browser ko sirf display karna hota hai

SSR ka process

  • User request bhejta hai server ko
  • Server data fetch karta hai
  • Server HTML page generate karta hai
  • Ready page browser ko send hota hai

SSR ke faide (Advantages)

  • Fast first load
  • SEO friendly hota hai
  • Low-end devices par bhi acha perform karta hai
  • User ko jaldi content dikh jata hai
