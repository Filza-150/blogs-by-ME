Client Side Rendering (CSR) kya hota hai?

Client Side Rendering ek technique hai jisme web page browser (client) par render hota hai. Server sirf basic HTML aur JavaScript bhejta hai, baqi content browser khud generate karta hai.

Simple:

  • CSR = page browser khud banata hai

CSR kyu use hota hai?

  • Dynamic aur interactive websites banane ke liye
  • Fast navigation (page reload ke bina)
  • Modern frameworks (React, Angular) mein use hota hai

Example:

Jab tum kisi React app ko open karti ho → pehle empty page load hota hai → phir JavaScript run hoti hai aur content screen par show hota hai

CSR ka process

  • Server basic HTML aur JS file bhejta hai
  • Browser JavaScript load karta hai
  • Data API se fetch hota hai
  • Page dynamically update hota hai

CSR ke faide (Advantages)

  • Smooth user experience
  • Fast page transitions
  • Highly interactive UI
  • Single Page Applications (SPA) ke liye best
