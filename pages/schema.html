Schema kya hota hai?

Schema ek structure hota hai jo define karta hai ke database mein data kis format mein store hoga.

Simple:

  • Schema = data ka blueprint ya structure

Schema kyu use hota hai?

  • Data ko organized rakhne ke liye
  • Data ki type aur format define karne ke liye
  • Invalid data ko rokne ke liye

Example:

Agar tum user ka data store kar rahi ho → schema define karega ke user ka naam string hoga, age number hoga, aur email required hoga

Schema ke main points

  • Fields
    Data ke attributes (jaise name, age, email)
  • Data Types
    Har field ka type (String, Number, Boolean)
  • Validation
    Rules jo ensure karte hain ke data sahi ho

Schema ke faide (Advantages)

  • Data consistency maintain hoti hai
  • Errors kam hote hain
  • Database structure clear hota hai
  • Development easy ho jati hai
