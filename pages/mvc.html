MVC kya hota hai?

MVC ek design pattern hai jo application ko 3 parts mein divide karta hai taake code organized aur easy to manage ho.

Simple:

  • MVC = code ko alag alag parts mein divide karne ka tareeqa

MVC kyu use hota hai?

  • Code ko clean aur organized rakhne ke liye
  • Maintenance aur debugging asaan banane ke liye
  • Large projects ko manage karne ke liye

Example:

Jab tum ek web app banati ho → user input deta hai → controller usko handle karta hai → model data manage karta hai → view user ko output dikhata hai

MVC ke 3 parts

  • Model
    Data aur database ko handle karta hai
  • View
    UI (user interface) dikhata hai
  • Controller
    Model aur View ke beech connection banata hai

MVC ke faide (Advantages)

  • Code reuse hota hai
  • Project structure clear hoti hai
  • Team work asaan hota hai
  • Scalable applications ban sakti hain
